# AI Cyber Audit Tool

Система комплексного аудиту безпеки, що використовує ШІ для аналізу цифрових активів.

## Функціональні можливості
- **Email Security:** Перевірка email-адрес на витоки даних.
- **Foundation Analysis:** Верифікація благодійних фондів та дональних посилань.
- **OSINT & Infrastructure:** Глибокий аналіз сайтів, серверів, відстеження відкритих портів та мережевої інфраструктури.
- **Security Validation:** Перевірка паролів на надійність та валідація даних платіжних карток.
- **Report Generation:** Автоматична генерація професійних PDF-звітів за результатами перевірки.

## Технологічний стек
- **Мова:** Python 3.x
- **Інтерфейс:** Streamlit
- **Аналітичне ядро:** Google GenAI SDK (Gemini)
- **Звітність:** ReportLab

## Встановлення та запуск
1. Встановіть залежності: pip install -r requirements.txt

2. Налаштуйте змінну середовища GEMINI_API_KEY з вашим ключем доступу.

3. Запустіть проєкт: streamlit run app.py
